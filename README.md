# Bank Manager System

A simple banking management system developed using Java and NetBeans.

## Features
- Create bank account
- Deposit money
- Withdraw money
- Transfer money
- Display account information

## Technologies Used
- Java
- NetBeans IDE
- Ant Build Tool

## Project Structure

```text
src/                Source code
nbproject/          NetBeans configuration
build.xml           Build configuration
manifest.mf         Manifest file
```

## How to Run
1. Open project in NetBeans
2. Click Run Project
3. Execute the main class
